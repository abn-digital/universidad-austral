# Taller de IA — Universidad Austral (Q1 2026)

Portal oficial para las clases de **Dashboarding + AI** en Ingeniería de la Universidad Austral. Este espacio sirve como centro de instrucciones, inspiración y plataforma de entregas para el proyecto final.

## 🎯 El Desafío: Creación de una Empresa Digital con IA
Los alumnos deberán imaginar una marca desde cero y materializarla usando herramientas de Inteligencia Artificial generativa. El objetivo es aprender a integrar activos (visuales, sonoros y de datos) en una narrativa coherente.

### 📦 Entregables
1. **Web (Landing Page)**: Propuesta de valor clara (v0.dev, Lovable, Bolt).
2. **Branding**: Logo y manual de marca (Midjourney, DALL-E).
3. **Audio**: Jingle oficial de la marca (Suno AI).
4. **Video**: Anuncio corto de producto (Veo / Runway).
5. **Dashboard**: Métricas clave del negocio.

---

## 🛠️ Stack Tecnológico del Portal
Este portal ha sido construido siguiendo los [Hike Brand Guidelines](https://github.com/abn-digital/guidelines-marca-hike).

- **Frontend**: Vite + Vanilla JS
- **Estilos**: Vanilla CSS (Hike Design System)
- **Hosting**: Firebase Hosting (`hike-agentic-playground`)
- **Backend**: Google Apps Script Webhook (para recolección de datos en Sheets)

## 🚀 Deploy
El portal se hostea automáticamente en:
[https://universidad-austral.web.app](https://universidad-austral.web.app)

Para realizar cambios manualmente:
```bash
npm install
npm run build
firebase deploy
```
