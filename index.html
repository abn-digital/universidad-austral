La Consigna

Qué tenés que entregar.

TL;DR: Inventá una empresa, dale identidad, construí su presencia digital y presentala con un video. Todo usando IA como herramienta de producción, no como atajo.

01 — Obligatorio

Manual de Marca

Logo, paleta de colores, tipografía y tono de voz. Tiene que ser un documento coherente. Si el logo es minimalista y el copy es informal, hay un problema.

PDF o Notion Logo en PNG
02 — Obligatorio

Landing Page

Una página web publicada con dominio real (gratuito está bien). Tiene que tener navegación, sección de producto/servicio y el video embedded. No vale una captura de pantalla.

URL publicada Responsiva
03 — Obligatorio

Video + Jingle

Un video de 30-45 segundos que presente la marca. Las IAs generan clips de ~8 segundos gratis — encadenán 4-5 clips en CapCut (gratis, online) y agreguen el jingle encima.

30-45 seg Editar en CapCut Audio incluido
04 — Opcional (+nota)

Dashboard de Negocio

Si quieren ir a fondo, armen un dashboard con métricas simuladas de su empresa: ventas proyectadas, CAC, LTV. Looker Studio conectado a un Google Sheet funciona perfecto.

Looker Studio Google Sheets

¿Cómo los evaluamos?

40% Cohesión de Marca

Logo, colores, tipografía y tono de voz tienen que parecer de la misma empresa en cada pieza.

30% Integración Técnica

Video embebido en la web, jingle corriendo, links funcionando. Si algo está roto el día de la presentación, baja la nota.

30% Claridad del Pitch

En 2 minutos tienen que poder explicar qué hace la empresa, para quién y por qué alguien pagaría por eso.

Bonus: Live Data

Conectá tu web a datos reales.

Requisito extra: Tu landing page tiene que tener una sección o tab que muestre datos en vivo desde una API pública. No hace falta que sea compleja — un card con el precio del dólar actualizado ya cuenta. Lo importante es que los datos vengan de una API real, no estén hardcodeados.

💵 Finanzas

DolarAPI

Cotización del dólar en Argentina: blue, oficial, MEP, CCL, cripto. Ideal si tu empresa es fintech, ecommerce o cualquier cosa con precios.

Sin API Key JSON 
fetch('https://dolarapi.com/v1/dolares')
🌤️ Clima

Open-Meteo

Clima actual y pronóstico de cualquier ciudad del mundo. Ideal para turismo, logística, agro o sustentabilidad.

Sin API Key Geolocalización 
fetch('https://api.open-meteo.com/v1/forecast?latitude=-34.6&longitude=-58.4&current_weather=true')
🌍 Datos Globales

REST Countries

Info de todos los países: población, bandera, moneda, idioma. Perfecto para dashboards comparativos o empresas de expansión internacional.

Sin API Key 250+ países 
fetch('https://restcountries.com/v3.1/name/argentina')
₿ Cripto

CoinGecko

Precio de Bitcoin, Ethereum y +10.000 criptos en USD y ARS. Útil para fintech, inversiones o cualquier web3.

Sin API Key Precio en ARS 
fetch('https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=usd,ars')

💡 ¿Encontraste otra API que encaja mejor con tu empresa? Usala. Cualquier API pública y gratuita es válida. Lo que evaluamos es que los datos se carguen en tiempo real desde el código, no que usen una de estas cuatro.

Prompt Engineering

Cómo hablarle bien a la IA.

La diferencia entre un resultado genérico y uno profesional no es la herramienta, es el prompt. Estas son las técnicas que más impacto tienen:

01

Asigná un Rol

Antes de pedir algo, decile a la IA quién tiene que ser.

❌ "Escribí un eslogan para mi marca de agua" ✅ "Sos un director creativo de una agencia de publicidad premium. Escribí 5 slogans para una marca de agua mineral enfocada en deportistas urbanos. Estilo: directo, minimalista, en inglés."
02

Describí el Contexto Completo

Para imágenes y videos, más detalle = mejor resultado.

❌ "Logo para empresa de tecnología" ✅ "Minimal logo for a B2B SaaS company called 'Nexora'. Icon only, no text. Abstract geometric shape suggesting connectivity and growth. Color palette: deep navy and electric blue. White background. SVG style."
03

Iterá, No Aceptes lo Primero

El primer resultado es siempre el más genérico. Pedí variaciones.

✅ "Mantené el estilo pero hacé el ícono 30% más simple y que el color principal sea más saturado" ✅ "Dame 3 variaciones: una más corporativa, una más joven y una más premium"
04

Chain of Thought para Estrategia

Pedile que piense antes de responder en tareas complejas.

✅ "Antes de responder, analizá: 1) el mercado objetivo, 2) la competencia, 3) la propuesta de valor. Luego escribí el pitch."
